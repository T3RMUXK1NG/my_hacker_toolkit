## ✨ Features

| # | Tool | Description |
|---|------|-------------|
| 1 | 🔰 **Hello World** | Test toolkit startup |
| 2 | 📁 **Directory Scanner** | Scan directories with HTML report generation |
| 3 | 👁️ **Directory Monitor** | Live directory monitoring (inotifywait / bash fallback) |
| 4 | 🌐 **Network Info** | IP, gateway, DNS, active connections |
| 5 | 📡 **Ping Scan** | Internet/network connectivity test |
| 6 | 🔄 **File Comparison** | Diff files and folders side-by-side |
| 7 | 🔍 **Duplicate Finder** | Find duplicate files using MD5 checksums |
| 8 | 🔒 **File Encryption** | AES-256-CBC encryption with OpenSSL |
| 9 | 🔓 **File Decryption** | Decrypt encrypted files |
| 10 | 🎨 **Theme Switcher** | Green / Neon color themes |
| 11 | 🌐 **Language Switcher** | Hindi / English language support |
| 12 | 🔄 **Version Check** | Check for updates from GitHub |
| 13 | 💾 **Auto Backup** | ZIP backup of entire toolkit |
| 14 | 📥 **Restore Backup** | Restore from ZIP backup |

### Key Highlights
- **Hindi-Friendly UI** — Full Hindi and English language support
- **Termux Compatible** — Works on Android with Termux (TTS, vibration alerts)
- **Color Themes** — Switch between Green and Neon themes
- **Logging System** — All actions logged with timestamps
- **Auto-Update** — Check for new versions from GitHub
- **Backup & Restore** — Full toolkit backup with ZIP compression

## 🚀 Installation

### Termux (Android)
```bash
pkg install bash git openssl zip
git clone https://github.com/rajsaraswati-jatavv/my_hacker_toolkit.git
cd my_hacker_toolkit
chmod +x my_hacker_toolkit.sh
./my_hacker_toolkit.sh
```

### Linux
```bash
git clone https://github.com/rajsaraswati-jatavv/my_hacker_toolkit.git
cd my_hacker_toolkit
chmod +x my_hacker_toolkit.sh
./my_hacker_toolkit.sh
```

## 📖 Usage

1. **Launch** the toolkit with `./my_hacker_toolkit.sh`
2. **Select** a tool (1-14) from the menu
3. **Follow** the on-screen prompts (in Hindi/English)
4. **Type 0** to exit

### Language Setup
```bash
# Hindi (default)
export TOOL_LANG=hi

# English
export TOOL_LANG=en
```
